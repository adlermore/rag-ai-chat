# 03 — Дизайн-система (shadcn/ui)

> Обязателен к прочтению перед любой работой над UI. Отклонения от токенов — только
> через обновление этого файла.

## Дизайн-концепция

Внутренний инструмент, которым люди пользуются каждый день, чтобы **быстро получить
проверяемый ответ**. Дизайн подчинён трём идеям:

1. **Доверие видно.** Источники — не сноска, а часть ответа. Уровень уверенности
   системы (уверенный / неуверенный / отказ) имеет явный визуальный язык.
2. **Тишина вокруг текста.** Ответ — главный контент. Интерфейс нейтральный,
   плотный, без декора; цвет появляется только там, где несёт смысл.
3. **Локальная идентичность без клише.** Палитра построена на двух ереванских
   материалах: тёплый серо-розовый **туф** (нейтрали поверхностей) и **абрикос**
   (единственный акцент). Это даёт продукту собственное лицо, оставаясь строгим
   enterprise-инструментом.

**Фирменный элемент (signature):** «цитатные якоря» — маленькие нумерованные чипы
⟨1⟩⟨2⟩ прямо в тексте ответа AI; при наведении/тапе открывается карточка источника
с названием документа, страницей и фрагментом текста. Это визуальное воплощение
главного обещания продукта: каждое утверждение можно проверить.

## Ребрендинг: Центральный банк Армении (2026-07-15)

Продукт визуально принадлежит **Կենտրոնական Բանկ** (ЦБ РА): фирменный глубокий
синий вместо абрикоса, холодные нейтрали вместо туфа, эмблема-портик с золотой
каймой (`apps/web/src/components/brand.tsx`, favicon `apps/web/src/app/icon.svg`).
Золото (`--brand-gold`) — ТОЛЬКО эмблема/декор, не интерактив. Название:
«ԿԲ Գիտելիքների օգնական», на логине — полное имя банка. Таблица ниже —
актуальные значения (историческая туф/абрикос-палитра заменена).

## Палитра (CSS-переменные для shadcn)

Светлая тема — основная. Тёмная — обязательна (сотрудники сидят в чате целый день).

| Токен | Light | Dark | Роль |
|---|---|---|---|
| `--background` | `#F7F9FC` | `#0F141C` | фон приложения (холодный почти-белый) |
| `--foreground` | `#1A2332` | `#E8ECF2` | основной текст |
| `--card` | `#FFFFFF` | `#151B25` | карточки, панели |
| `--muted` | `#EDF1F7` | `#1D2530` | вторичные поверхности |
| `--muted-foreground` | `#5C6675` | `#93A0B4` | вторичный текст |
| `--border` | `#DEE4EC` | `#2A3442` | границы |
| `--primary` | `#17427C` | `#6F9CE0` | **синий ЦБ** — кнопки, ссылки, активные состояния |
| `--primary-foreground` | `#FFFFFF` | `#0D1524` | текст на primary |
| `--ring` | `#17427C` | `#6F9CE0` | focus ring |
| `--brand-gold` | `#C9A227` | `#D9B545` | золото ЦБ — эмблема и декор, НЕ интерактив |
| `--destructive` | `#B3261E` | `#E5544B` | удаление, ошибки |

Семантика уверенности ответа (не из палитры shadcn — дополнительные токены):

| Токен | Light | Роль |
|---|---|---|
| `--confidence-high` | `#1E7F4F` | уверенный ответ (без визуальной метки) |
| `--confidence-low` | `#B07D10` | янтарная: бейдж «ответ может быть неполным» |
| `--confidence-none` | `#6E675F` | серая: отказ (информации нет) |

Правило: синий ЦБ — только для интерактива (одна главная кнопка на экран, ссылки,
активный пункт меню, цитатные чипы). Не использовать для фонов и декора.
Уверенность больше НЕ показывается цветной левой кромкой (убрана — «тишина вокруг
текста»). Сигнал только в **бейдже**: `low` → компактный янтарный бейдж с иконкой
сверху ответа; `high` → без метки; отказ — отдельное спокойное сообщение.
Каждый ответ AI помечен эмблемой-аватаром банка слева (идентичность ассистента).

## Типографика — армянский язык

Это главный технический пункт дизайна: **Inter, Geist и большинство модных шрифтов
не содержат армянских глифов.** Использовать:

- **Noto Sans Armenian** (Google Fonts, variable 100–900) — весь UI и текст ответов.
- **Noto Serif Armenian** — только крупные заголовки экранов (display), даёт
  характер без потери читаемости.
- `JetBrains Mono` — числа в аналитике, ID, технические значения (арм. глифы
  не нужны — там латиница/цифры).

```ts
// next/font
import { Noto_Sans_Armenian, Noto_Serif_Armenian } from "next/font/google";
const sans = Noto_Sans_Armenian({ subsets: ["armenian"], variable: "--font-sans" });
const serif = Noto_Serif_Armenian({ subsets: ["armenian"], weight: ["600","700"], variable: "--font-serif" });
```

Шкала: display 28/36 serif-600 · h2 20/28 sans-700 · body 15/24 sans-400 ·
small 13/20 · caption 12/16. Армянский текст в среднем длиннее английского на
20–30% — все компоненты проектировать с запасом ширины, никаких фиксированных
ширин под "английскую" длину строк.

## Установка shadcn

```bash
pnpm dlx shadcn@latest init   # style: default, baseColor: neutral, cssVariables: true
pnpm dlx shadcn@latest add button input textarea card dialog sheet dropdown-menu \
  table badge avatar skeleton toast tooltip tabs select scroll-area separator \
  alert alert-dialog progress command popover form sonner
```

Переменные из палитры выше кладутся в `globals.css` (`:root` / `.dark`).
Радиус: `--radius: 0.625rem`. Обёртки и композиты — в `packages/ui`.

## Ключевые экраны и паттерны

### Чат клиента (главный экран)

```
┌────────────┬──────────────────────────────────────────────┐
│ Sidebar    │  Диалог                                       │
│ (Sheet на  │  ┌─ Сообщение пользователя (справа, muted)    │
│  мобиле)   │  ◉ Ответ AI (аватар-эмблема + card):          │
│ история    │  │  [янтарный бейдж, только если low]           │
│ чатов,     │  │  Текст ответа… факт ⟨1⟩ … данные ⟨2⟩        │
│ поиск,     │  │  ── Источники ──────────────────────        │
│ + новый    │  │  [карточки источников]                       │
│            │  │  ⟨1⟩ Կանոնակարգ.pdf · էջ 12   [открыть]     │
│            │  │  ⟨2⟩ Սակագներ.xlsx · Թերթ 2, տող 14         │
│            │  └──────────────────────────────────────       │
│            │  [Composer: textarea, autosize, ⌘↵, Send]      │
└────────────┴──────────────────────────────────────────────┘
```

- Ответ стримится (SSE) с курсором-мигалкой; источники дорисовываются в конце.
- Цитатный чип ⟨n⟩: `Badge` variant=outline, tabular-nums, primary-цвет; hover →
  `HoverCard` с фрагментом чанка; клик → скролл к карточке источника.
- Отказ («информации нет») — не выглядит как ошибка: спокойная серая карточка
  с иконкой поиска и советом переформулировать + примеры вопросов.
- Пустой чат — не пустота: 3–4 кликабельных примера популярных вопросов
  (из аналитики), заголовок serif-ом.

### Админ-панель

- Layout: постоянный sidebar (`Documents / Clients / Analytics / Audit`),
  breadcrumbs, `Command`-палитра (⌘K) для быстрых переходов.
- **Documents:** `Table` с колонками имя/тип/статус/чанки/обновлён; статус —
  `Badge` (queued=muted, processing=amber+`Progress`, ready=green, failed=red
  с текстом ошибки в `Tooltip`). Загрузка — dropzone-зона + `Progress`;
  индексация видна live (SSE). Замена/удаление — через `AlertDialog` с
  предупреждением об инвалидации кэша.
- **Analytics:** карточки-метрики (`Card` + JetBrains Mono для чисел),
  график вопросов по дням (recharts, один цвет — primary), таблица популярных
  вопросов, отдельно — **процент отказов** (это метрика здоровья базы знаний:
  если высокая — админу нужно догрузить документы).

### Состояния — обязательные для каждого экрана

Для каждого списка/экрана реализуются 4 состояния: loading (`Skeleton`,
повторяющий геометрию контента), empty (что это + кнопка действия), error
(что случилось + как исправить + retry), success. Тексты — по правилам ниже.

## Микротексты (армянский)

- Активный залог, глагол = действие: «Բեռնել փաստաթուղթ», не «Հաստատել».
- Ошибка = что случилось + что делать; без извинений и без «что-то пошло не так».
- Название действия не меняется по ходу: кнопка «Հրապարակել» → тост «Հրապարակված է».
- Все строки — в i18n-словаре `hy.json`, ключи английские (`chat.send`, `docs.upload`).

## Доступность и качество

- Видимый focus ring (абрикос) на всех интерактивных элементах; полная работа
  с клавиатуры (композер: ↵ отправить, Shift+↵ перенос).
- Контраст текста ≥ 4.5:1 (палитра выше проверена).
- `prefers-reduced-motion` уважается; анимации — только 150–200ms fade/slide
  на появление сообщений и панелей, ничего декоративного.
- Мобильная версия: чат — полноэкранный, sidebar → `Sheet`; админка —
  desktop-first, но таблицы горизонтально скроллятся.
- Ни одного инлайн-hex в компонентах — только токены.
